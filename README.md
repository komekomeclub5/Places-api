# Places-api

## 概要

台風の時に周辺のスーパーマーケットの営業状態をひと目でわかるようにしたい。

## 事前準備

DockerDesktopを事前にダウンロードしてください。


[DockerをMacにインストールする（更新: 2019/7/13）](https://qiita.com/kurkuru/items/127fa99ef5b2f0288b81)

## 使用方法

起動
```
docker-compose up -d
```

ブラウザでアクセス
```
localhost:8000
```

終了
```
docker-compose down
```
